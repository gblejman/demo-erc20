An all vanilla erc20 and faucet implementation

- monorepo + typescript
- dapp: nextjs + ethers
- contracts: hardhat + ethers + typechain. Contract abis and ts types exported to be shared in dapp

# Folder Structure

- /: workspace root
- /packages/dapp: frontend dapp
- /packages/harhat: contracts dev
